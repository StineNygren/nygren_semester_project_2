## Description
A online auction site were you can

- Search trough auction items
- Create your owne profile
- Bid on items
- Create an auction on an item

## Built With

- Java script
- Tailwind

## Getting Started

1. Clone the repo:

```bash
git clone https://github.com/StineNygren/nygren_semester_project_2.git
```

2. Installing:
```
npm i
```

3. Build:
```
npm run build
```
